# Personal Income & Expense Tracker

A simple, database-free personal finance application with AI-powered natural language entry.

## Features
- **Dashboard**: View Daily, Monthly, and Yearly summaries.
- **Charts**: Visual breakdown of expenses by category.
- **Manual Entry**: Add income and expenses via a simple form.
- **AI Chatbot**: Add transactions using natural language (English, Bangla, or Mixed).
  - Example: "Spent 450 taka for dinner"
  - Example: "Ajke 500 taka income holo"
  - Example: ""450 টাকা বাজারে খরচ করেছি""
- **Data Persistence**: All data is saved safely in local JSON files
